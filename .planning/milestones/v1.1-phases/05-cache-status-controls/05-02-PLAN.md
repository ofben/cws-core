  <name>Task 1: Add cache status rendering to render_settings_page()</name>
  <files>includes/class-cws-core-admin.php</files>
  <action>
    Add a private helper method render_cache_status_block() to CWS_Core_Admin, then call it inside render_settings_page() immediately before the Clear Cache button.

    render_cache_status_block() logic:
    - Read $last_time = get_option( 'cws_core_last_fetch_time', 0 );
    - Read $last_status = get_option( 'cws_core_last_fetch_status', null );
    - If $last_time is empty (0 or false): output no-cache state (see below)
    - If $last_time exists: output status state (see below)

    No-cache state HTML (ID is used by JS in Task 2):
    <div id="cws-core-cache-status" style="margin-bottom:8px; color:#646970; font-size:13px;">
        <?php esc_html_e( 'No cache — will refresh on next page load', 'cws-core' ); ?>
    </div>

    Status state HTML (success — HTTP 200):
    <div id="cws-core-cache-status" style="margin-bottom:8px; font-size:13px;">
        - Relative age: human_time_diff( $last_time, time() ) . ' ago'
        - Absolute timestamp: wp_date( 'Y-m-d H:i', $last_time )
        - Primary text: "Last refreshed: {age} — HTTP {code}"
        - Secondary line (grey, font-size:12px): "(2026-03-03 14:22)"
    </div>

    Failure state HTML (HTTP code is not 200 and not 0):
    - Same wrapper, but color:#d63638 on the primary line
    - Text: "Last attempt failed — HTTP {$last_status}"
    - Absolute timestamp below in grey

    Connection error state (status = 0):
    - Same as failure state, color:#d63638
    - Text: "Last attempt failed — connection error"
    - Absolute timestamp below in grey

    Determination: if (int)$last_status === 200 → success. If (int)$last_status === 0 → connection error. else → HTTP error.

    Insert the render_cache_status_block() call in render_settings_page() by adding it after the paragraph tag before the Clear Cache button:
    After: <p><?php esc_html_e( 'Manage cached data and view cache statistics.', 'cws-core' ); ?></p>
    Add:   <?php $this->render_cache_status_block(); ?>
