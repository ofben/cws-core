Remove the 2,494-line virtual CPT infrastructure completely. Delete the class file, all 13 root
test/debug scripts, all 3 templates, and surgically excise every reference from the three files
that wired the virtual CPT in (cws-core.php, class-cws-core.php, class-cws-core-admin.php).

panel renders without the Test Virtual CPT section.
